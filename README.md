# 🚀 Gravity Search UI

A modern Google Gravity-inspired interactive UI experiment built with HTML, CSS, JavaScript, and Matter.js physics engine.

---

## ✨ Overview

Gravity Search UI transforms a traditional search page into a real-time interactive physics playground.

Instead of static UI elements, the interface behaves like physical objects affected by gravity, collisions, motion, and user interactions.

This project focuses on:

* frontend engineering concepts
* DOM manipulation
* physics simulation
* rendering synchronization
* interactive UI systems
* animation architecture

---

## 🎯 Features

* 🎨 Google-inspired modern UI

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Nidia-soys/gravity-search.git
```

Open the project folder:

```bash
cd gravity-search-ui
```

Run with Live Server or open `index.html` directly in the browser.
